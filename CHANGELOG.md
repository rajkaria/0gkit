# Changelog

This is the repository-level changelog. Per-package changelogs are generated by
[changesets](https://github.com/changesets/changesets) and live in each
`packages/0gkit-*/CHANGELOG.md`.

## 0.1.0 — Initial public release

First public release of **0gkit**, the neutral 0G builder toolkit, extracted
into its own repository and published to npm under `@foundryprotocol/0gkit-*`:

- `@foundryprotocol/0gkit-core` — network presets, viem client, `Receipt`, error taxonomy
- `@foundryprotocol/0gkit-chain` — explorer URLs, balance, receipts, faucet
- `@foundryprotocol/0gkit-storage` — upload / download / computeRoot / exists
- `@foundryprotocol/0gkit-compute` — provider discovery, broker inference, OpenAI shim
- `@foundryprotocol/0gkit-da` — data availability publish / verify
- `@foundryprotocol/0gkit-attestation` — TEE attestation parse / sign / verify
- `@foundryprotocol/0gkit-cli` — the `0g` command line
- `@foundryprotocol/0gkit-mcp` — every primitive as an MCP tool
- `@foundryprotocol/0gkit-react` — `useUpload` / `useDownload` / `useInference` / `useAttestation`

Every package is independently installable, neutral (no app/framework
dependency), and ships with a docs site and starter templates.

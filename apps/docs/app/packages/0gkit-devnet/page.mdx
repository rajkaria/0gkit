# 0gkit-devnet

Local 0G devnet harness used by `0g dev`.

## Install

```bash
pnpm add @foundryprotocol/0gkit-devnet
```

## Exports

- `DEFAULT_DEV_MNEMONIC`
- `VERSION`
- `clearState`
- `deriveAccounts`
- `detectAnvil`
- `isRunning`
- `readState`
- `spawnAnvil`
- `startComputeMock`
- `startDaMock`
- `startDevnet`
- `startStorageMock`
- `stopDevnet`

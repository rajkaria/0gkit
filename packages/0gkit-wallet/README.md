# @foundryprotocol/0gkit-wallet

Neutral wallet abstraction for 0G — implements the `Signer` interface from `@foundryprotocol/0gkit-core` with key loaders (`fromPrivateKey`, `fromFile`, `fromEnv`, `fromKMS`) and SIWE (Sign-In with Ethereum) helpers. Node/universal, RSC-safe, tree-shakable. Usage examples will be added in subsequent tasks.
